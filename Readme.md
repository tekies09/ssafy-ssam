## 📌 Intro

### 서비스 배경

- 코로나 이슈가 아니여도 점점 식어가는 야구의 인기..

- MZ 세대들은 보는 스포츠보다는 하는스포츠를 더선호..

- 야구의 전성기도 되찾고 MZ세대들에게 컨텐츠를 제공하는 야구 시뮬레이션 플랫폼으로 두마리의 토끼를 다 잡자!

### 서비스 목표

#### 나만의 라인업을 통해 야구 시뮬레이션을 진행하고 야구 신규 팬의 유입을 활성화
